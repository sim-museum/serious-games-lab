Always best to make a copy of the file before you start editing so you can go back.

Normally I set the AI speed in the track.ini in the track folder you are using.

[ GP ]
dlong_speed_adj_coeff =

reducing "dlong_speed_adj_coeff" will slow the AI. Increase will speed up.
Try adjusting in 0.01 intervals.

For race length I normally adjust the season file that has the track.
If you only have 67season it will be in there.

example

[Eventxx]
name=Grand Prix d'Albi
shortname=GP d'Albi
trackShortName=Albi
trackDirectory=albi67
flag=franflg
month=7
day=13
numberOfLaps=80
startingGrid=5

80 will be full Grand Prix length.
Long will be 30% distance = 24 laps
Short will be 15% distance = 12 laps
novice will be 8% = 7 laps

If you reduced the GP race length to 40 laps you would get a 3 or 4 laps race in novice mode.
