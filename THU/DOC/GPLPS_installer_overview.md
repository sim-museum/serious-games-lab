## **Overview**
- The GPLPS GPL Installer is a modernized installation tool for **Grand Prix Legends**, a classic racing simulation.
- It incorporates updates, patches, and add-ons to make the sim compatible with modern hardware and enhance the gaming experience.
- Key benefits include:
  - Updated tracks and cars.
  - Improved graphics rasterizers.
  - Community patches for online multiplayer and additional features like long-track support.
  - Basic patches for sound and gameplay optimization.
  - Simplified installation process compared to the original CD or GPL 2004 Demo.

---

## **Preparation for Installation**
1. **Download the Installer**:
   - Obtain the latest version from [http://gplps.wordpress.com/gplps-gpl-installer/](http://gplps.wordpress.com/gplps-gpl-installer/).

2. **Preparation**:
   - Insert the GPL CD into your drive but close the auto-start window that may appear.

---

## **Installation Process**
The installer’s intuitive process guides users through various steps. Key points to remember:

### **Installer Permissions**:
- When prompted, allow the installer to make changes to the hard disk. This is required to copy files.

### **Language Selection**:
- This sets both the installation language and the language used in the game. There is no way to go back to this screen, so choose carefully.

### **Destination Directory**:
- Avoid choosing system folders like `C:\Program Files\GPL` due to Windows security restrictions.
- Recommended paths: `C:\Sierra\GPL` or `D:\Sierra\GPL`.

### **Component Selection**:
- The installer allows customization of components to be installed. The default setup usually works for most users, but individual options are explained below.

## **Components Overview**
The installer offers flexibility in customizing the installation. Below are the details of each component:

### 1. **Core GPL**:
   - **Size:** 267.5 MB
   - Transfers files from the CD to the target directory.

### 2. **Basic Patching (1.2.0.2)**:
   - Provides the latest official Papyrus patches.

### 3. **Community GPL Patch 1.3**:
   - **Size:** 17.2 MB
   - Adds new features like a 1024-track limit, setup fine-tuning, and long-track support (e.g., Targa Florio).
   - No changes to physics.

### 4. **Sound Add-ons**:
   - Includes enhanced sound files by Keeper and ducwolf.
   - Mono sounds are generally recommended for better compatibility with the GPL sound engine.

### 5. **Video Rasterizers**:
   - Modernized **D3Dv2** and **OpenGLv2** rasterizers for improved graphics stability on newer systems.
   - Avoid using older rasterizers unless necessary.

### 6. **GPLEA Cars**:
   - Graphically improved car models for various 1967 F1 cars (e.g., Ferrari, Lotus, Honda).
   - Optional activation of 2D helmets for easier identification in online races.

### 7. **Standard Car Cameras**:
   - Restores the original Papyrus camera angles.

### 8. **Track Updates**:
   - Updated versions of 1967 F1 season tracks like Kyalami, Monaco, Nürburgring, and others.
   - Features historic and online-compatible variants.

### 9. **Season Grid Position Fix**:
   - Corrects the starting grid positions for the 1967 F1 season.

### 10. **BAPOM Program and Map Pack**:
   - Adds visually appealing program menus and track maps.

### 11. **Smoke Effects**:
   - Two smoke effect options by MClark and Jay Beckwith.

### 12. **Sparks**:
   - Enhanced spark effects by Jay Beckwith.

### 13. **Corner Worker and Flagman**:
   - Visual updates for track corner workers and flag signals.

### 14. **Pitboard**:
   - Updated pitboard font for better readability.

### 15. **Teaminfo Badges**:
   - Corrects team names and badges (e.g., Honda and Cooper).

### 16. **1967 Driver Teams**:
   - Updates driver rosters, including corrections like adding Jackie Stewart.

### 17. **Repository of Updates**:
   - All optional components are stored in a repository directory, allowing reinstallation later without downloading again.

## **Post-Installation Setup**
After installation, several essential setup steps must be completed before playing:

### 1. **First Start and Reboot**:
   - Restart your computer after installation to finalize changes.

### 2. **Locate Shortcuts and Program Group**:
   - Check for desktop icons and the GPL program group for easy access.

### 3. **Wheel Calibration**:
   - Ensure your wheel, pedals, or gamepad are recognized by Windows. Calibrate them within GPL.

### 4. **GPL Welcome Screen**:
   - Input your first and last name (avoid special characters) and save the entries.

### 5. **Graphics Settings**:
   - Adjust resolution, detail bias, and enable features like full mirror detail and trackside objects.
   - Set the rasterizer to the version chosen during installation (e.g., D3Dv2 or OpenGLv2).

### 6. **Sound & Gameplay Settings**:
   - Disable driving aids for a more realistic experience. Defaults are generally fine for gameplay.

### 7. **Leave GPL**:
   - Exit GPL after setting up to ensure all configurations are saved and activated.

## **Starting a Race**
1. Launch GPL again and choose **Single Race** from the menu.
2. Select a track and adjust race settings.
3. Start the session by proceeding through the menus until you are in the car.

---

## **Troubleshooting and Other Notes**
1. **No Cursor on First Launch**:
   - Use the TAB key to navigate fields and input your name.

2. **Recommended Training Settings**:
   - For practice, choose the **Training Option** and the **Grand Prix** car type.
   - Use **Shift + R** to reset the car as needed.

3. **Additional Help Documents**:
   - **Credits.pdf**: Lists contributors to GPL add-ons.
   - **GPLEA.pdf**: Details customization for GPLEA cars.
   - **RasterisersAndPribluda.pdf**: Covers rasterizer configuration and Pribluda (on-screen display for data like sector times and tire temperatures).

---

## **Important Tips**
- Always reboot after installation to finalize settings.
- Exit GPL after making setup changes to activate them.
- Use the default installation options unless you have specific preferences or experience with GPL modifications.
