INSTALLATION


1/ Manually install the GEMP2.exe update ( to Program files/GPLSecrets/Gem+ ) which will update Gem+ to version 2.5.32.
This Gem+ update is essential for the 67 Patch to work.

2/ Run the 67 Patch installer.
Check the install path. The default install path is C:\Sierra\GPL.
IF YOUR GPL INSTALL HAS A DIFFERENT PATH YOU MUST CHANGE THE PATH WHEN PROMPTED.

3/Remove the gplc67.exe from your gpl folder.

4/Start Gem+ and hit the gpl button or exit Gem+ to create a new patched gplc67.exe
