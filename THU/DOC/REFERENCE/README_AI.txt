GPL 60 fps patch test release v1, May 2009

by Petteri Pajunen and Nigel Pattinson

IMPORTANT

Please make sure you read the instructions related to 60fpsv1 patch. This readme only explains the AI part.

Don't report things like wrong laptimes or erratic AI behaviour in general, since these are known problems. If you have a bug report not related to 60 fps AI, but related to 60 fps, post it at SRMZ.net in the 60 fps release topic.

If you do find 60 fps AI problems, please report them at SRMZ.net in the 60 fps AI patch release topic.

This patch is not finished, and is released mainly to allow people to easily fine tune their GPL install to get solid 60 fps when other cars are on track. The 67 AI is somewhat driveable (and could be even more so by editing the gpl_ai.ini).

If you have lots of experience with tweaking AI, consider modifying the gpl_ai.ini to overcome bad AI behaviour. If you think you find something which significantly improves the AI, please report it.



INSTALLATION

Put 60fpsaiv1.xml in the GEM+ Options folder, back up your gpl_ai.ini file, and copy gpl_ai_60fps.ini over gpl_ai.ini when you want to test 60 fps AI. Do not edit track.ini values. 


USAGE

Select 60FPSAIV1 in GEM+ *and* unselect any other 60 fps patches. See the README for 60 fps V1 patch for more instructions.
