INFOS:

This track has a complete new layout from the previous creation for
GPL, I recreated the 1967 Sebring 12 hours.  I did a lot of research
to find out how exactly was the layout in '67, I did my best to
recreate it the most accurate as possible, keeping an eye to the
performance on the lower systems.  A few leftover .3do and .mip are
still from the older sebring 1959 by Smogbike, credits to the original
authors.

You find 3 .jpg files for GEM+: sebr67-GT.jpg : the GT version
(default) sebr67-F1.jpg : F1 version sebr67.jpg : the one which is
actually used by GEM+, rename to this whatever you like of the above


The track is a mandatory addon if you want to do the championship with
the SportCar MOD.
