GPLPS Track Installer for F1 1965 season

This installer will install the missing tracks Clermont-Ferrand 65,
East London and Charade Clermont-Ferrand (For GPL Rank 65 V1). Also it
includes Goodwood, Syracuse, Enna, Brands Hatch and Killarne. All
credits belong to the original track developers.

Status: beta

Link: http://gem.grandprixlegends.info/GPL65ModTracks_0.5.exe

Setup type: Normal GPL trackpack installation

This installer basically installs a complete GPL 1965 F1 Mod Version
2.0.2. To allow a one-click installation this installer is a merged
installation of the original GPL 1965 F1 Mod Version 2.0.1, the high
res version of the Lotus 33 and the later published Version
2.0.2. Additional the recommended steering wheel fix from MECH is
included as well. You will find the 65mod raised add-on from MECH in
the repository\GPL65\MECH folder as well as an option in the
Select Components. You have the choice to
select the original, full installation, a customized, a historic ride
height or a historic ride height with 2D helmets. Theres
still some interesting debate as to the ride height of the original
mod or the changed one from MECH is the right one. Due to the zillion
options Ill leave that to you and so I have limited the
options to basically 2 when you want to use the ride height fix from
MECH.  Just to add that the ride height fix from MECH is included as
well as an AI XML option for GEM+ which is kindly provided by maddog!
