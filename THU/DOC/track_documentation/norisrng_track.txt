
General Info: ------------- To install, unzip the contents of this zip
file to a TEMP folder and run 'trackinstall.exe', the track should
install automatically.

This install is ready for GEM+ integration.  It will utilize the
67season.ini editing capabilites of this great tool created by Paul
Thurston.  To use this track with GEM+ you must copy two files from
the install archive after installing the track.

Copy the files tgem.ini and screenshot.bmp to the norisrng folder
after install.


For Manual Install: --------------- Create folder in
\sierra\gpl\tracks called norisrng

unzip all files in archive to \sierra\gpl\tracks\norisrng folder

edit \sierra\seasons\67season.ini and \sierra\seasons\gp.ini by adding
the following

[EventXX] name=Nuernberg Grand Prix shortname=Norisring
trackDirectory=norisrng month=5 day=14 numberOfLaps=128 startingGrid=4

Dont forget to increment the numEvents=XX line at the top of
each... and make sure the XX's agree.
