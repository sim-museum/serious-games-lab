**Section 3: Creating a Pilot Profile**
---------------------------------------

**3.1. Setting Up a Pilot**
---------------------------

*   Navigate to the "Logbook" section in the main menu.
*   Steps:
    
    1.  Select "New" to create a new pilot profile.

**Section 4: Configuring Controls**
-----------------------------------

**4.1. Setting up the Joystick**
--------------------------------

*   Access controls under `Setup > Controllers`.
*   **Joystick Axes:**
    
    *   Configure pitch, roll, and other axes under "Advanced > Flight Control."
    *   Assign joystick axes for rudder pedals or mini sticks as needed.
    

**4.2. Key Mapping**
--------------------

*   Falcon BMS uses a default key file (`BMS.key`).
*   To customize controls:
    
    1.  Find the desired command in the key bindings list.
    2.  Click on the left of the command and press the desired joystick or keyboard input to remap it.
    3.  Save the changes as a new key file (e.g., "MyKeyFile").
    
*   Tip for Identifying Key Commands:
    
    *   If a command is mentioned in a tutorial (e.g., "TMS Left"), you can look it up in the controller setup to find the associated key.

**Section 6: Starting the F-16**
--------------------------------

**6.1. Pre-Start Procedures**
-----------------------------

1.  **Power Systems:**
    
    *   Set the main power switch to "Main Power" on the electronics panel.
    
2.  **Fuel Systems:**
    
    *   Ensure the master fuel switch is on and the engine feed is set to "Normal."
    
3.  **Air Source:**
    
    *   Set the air source knob to "Normal."
    
4.  **Lights:**
    
    *   Turn on navigation lights for safety.
    

**6.2. Engine Startup**
-----------------------

1.  Engage the **Jet Fuel Starter** (JFS) to "Start 2."
2.  Wait until the RPM gauge reaches ~25%.
3.  Move the throttle slightly forward and press `Alt+I` to pass the throttle detent.
4.  Wiggle the throttle if necessary until the RPM increases, then pull the throttle back to flight idle.

**6.3. Avionics Startup**
-------------------------

1.  Flip all avionics power switches to "On."
2.  Turn the INS (Inertial Navigation System) to "Norm" for alignment (takes ~8 minutes).
3.  Enable radar systems:
    
    *   Switch radar power on.
    *   Radar altimeter to "Standby."
    
4.  Turn on the HUD (Heads-Up Display):
    
    *   Adjust brightness using the HUD wheel.
    

**6.4. Countermeasures**
------------------------

1.  Activate electronic systems:
    
    *   Turn on the jammer and RWR (Radar Warning Receiver).
    *   Set the countermeasures mode to "Manual."
    *   Use Program 1 as default.
    

**6.5. Ejection Seat**
----------------------

*   Arm the ejection seat.

Study Guide for Ramp Start and Operation Procedures for the F-16
----------------------------------------------------------------

**Section 1: Ramp Starting the F-16**
-------------------------------------

1.  **Activating the Radar Warning Receiver (RWR):**
    
    *   Press the **power button** for the RWR located near the left panel.
    
2.  **Setting Up Communications:**
    
    *   Turn the **TACAN knob** to "UFC" (Up-Front Controls).
    *   Turn on the **Electronic Countermeasure (ECM)** switch.
    *   Adjust the **radio audio levels** using the knobs (set them to your preferred volume for each radio).
    *   Set **radio transmit** to "Both."
    
3.  **Using Game Time Acceleration:**
    
    *   Speed up game time by pressing **Shift + Tab** (up to 64X).
    *   Slow it back down to normal (1X) using **Shift + Caps Lock** repeatedly.
    
4.  **INS Alignment:**
    
    *   When the INS finishes aligning, click to set it to **"NAV" mode** on the INS panel.
    
5.  **Returning to Default UFC Menu:**
    
    *   Press the **"Return" button** on the UFC to bring back the default menu, which displays:
        
        *   UHF channel
        *   VHF channel
        *   Current steer point
        
    

**Section 2: Loading and Customizing the Data Cartridge**
---------------------------------------------------------

1.  **Loading the Data Cartridge:**
    
    *   **Before starting a mission:**
        
        *   Go to the game's map screen and click the **"Data Cartridge"** button in the bottom right.
        
    *   Customize the data cartridge settings, such as:
        
        *   **EWS Programs**: Define chaff and flare countermeasure programs.
        *   **MFD Layouts**: Set primary, secondary, and tertiary displays for different flight modes (e.g., NAV, A/G).
        *   **Radio Presets**: Assign frequencies to preset channels (e.g., UHF 15 = Tower).
        
    *   Save **customizations** before starting.
    *   **In the jet:**
        
        *   Press the corresponding MFD button to access the DTE (Data Transfer Equipment), then hit **"Load"**.
        
    
2.  **Key Customization Features:**
    
    *   Chaff/Flare Programs:
        
        *   Configure up to 4 countermeasure settings (e.g., Program 1: 1 flare, 0 chaff).
        
    *   Master Modes:
        
        *   Customize MFD (Multi-Function Display) layouts for different modes (e.g., radar on left, SMS on right in NAV mode).
        
    *   Preset Radio Channels:
        
        *   Assign the tower frequency to UHF 15 (use **"Set Tower"** for automatic assignment).
        
    

**Section 3: Takeoff Procedures**
---------------------------------

1.  **Line-Up:**
    
    *   Turn off **nosewheel steering** after aligning on the centerline to prevent accidental oversteer during takeoff.
    
2.  **Throttle Settings:**
    
    *   Use **Full MIL (Military Power)** or **Afterburner**, depending on the runway length and payload.
    *   Rotate at a speed of **160–170 knots**.
    *   Retract landing gear before reaching **300 knots** to avoid damage.
    
3.  **HUD Basics for Navigation:**
    
    *   **Flight Path Marker (FPM):** Align the FPM with the HUD "balloon" to fly toward the current steer point.
    *   Use **"S"** and **"Shift + S"** to cycle through steer points.
    *   **HUD Indicators:**
        
        *   Bottom left: Bearing/Distance from Bullseye.
        *   Top left: G-forces.
        *   Top right: Mach number.
        
    

**Section 4: Up-Front Controls (UFC) and HUD Key Functions**
------------------------------------------------------------

1.  **Radio Operations:**
    
    *   **COM 1 (UHF):**
        
        *   Input the preset channel (e.g., 15) or manual frequency.
        
    *   **COM 2 (VHF):** Same process as COM 1 for a second radio.
    
2.  **TACAN Navigation:**
    
    *   Press **1 (TILS)** on UFC.
    *   Input the channel (e.g., 11) and toggle between Yankee (0, Enter) or X-Ray.
    *   Use **SEQ** to switch between TACAN AA (Air-to-Air) or TR (Ground).
    
3.  **Altitude Alerts:**
    
    *   Use the UFC to set altitude alert thresholds:
        
        *   **Allow/Set Altitude:** E.g., set low altitude warning (e.g., 500 ft) or high altitude (e.g., 10,000 ft).
        
    
4.  **Steer Point Selection:**
    
    *   Manually set the steer point by pressing **4** on the UFC, typing the number, and hitting **Enter**.
    
5.  **Cruise Mode and Range Calculations:**
    
    *   Access **Cruise Page** via the UFC for fuel efficiency:
        
        *   **Range Page:** Displays fuel estimation for arrival at the current steer point.
        *   Optimal Cruise Settings:
            
            *   Determine the best altitude and Mach speed for fuel efficiency.
            *   Highlight the optimal speed on the HUD using **M Select (0)** while on the Cruise Page.
            
        
    
6.  **Bingo Fuel Warning:**
    
    *   Set the Bingo fuel threshold in the UFC:
        
        *   Press **List (2)** and input the fuel level (e.g., 1,500 lbs), hit **Enter**.
        
    

**Section 5: Additional Operational Notes**
-------------------------------------------

1.  **Using Game Features to Assist:**
    
    *   Request tower clearance to remove chocks by pressing **U** and then selecting option **8**.
    
2.  **HUD Navigation Tips:**
    
    *   **Steer Point Range:** Displayed on the bottom left of the HUD.
    *   Use the HUD "diamond" for visual alignment with steer points.
    
Study Guide: F-16 Flight Operations and Combat Systems
------------------------------------------------------


1. **Landing Procedure**
-------------------------

Key Steps:
----------

1.  **Reduce Speed Below 300 Knots:**
    
    *   Essential before lowering the landing gear; exceeding this speed could damage the gear.
    
2.  **Gear and Flaps:**
    
    *   Gear deployment automatically lowers the flaps.
    *   Use gear to slow down further.
    
3.  **Approach Speed:**
    
    *   Maintain approximately **200 knots** during final approach; reduce to **180 knots** as you near the runway.
    
4.  **HUD Changes:**
    
    *   A dashed 3-degree glide slope line appears on the HUD.
    *   Align this line with the runway’s 1000-foot markers to stay on the ideal path.
    
5.  **Landing Speed:**
    
    *   Aim to touch down at approximately **160 knots**.
    
6.  **Final Adjustments:**
    
    *   Over the runway threshold, aim the flight path marker at the horizon.
    *   Reduce throttle to idle and let the plane naturally settle onto the ground.
    
7.  **Aerobraking:**
    
    *   Keep the aircraft’s nose up to use aerodynamic braking.
    *   On shorter runways, lower the nose quickly and apply wheel brakes to avoid overrunning.
    

2. **Radar Basics**
--------------------

General Principles:
-------------------

*   **Cone-Shaped Scan Area:**
    
    *   The radar scans a cone in front of the aircraft. The scan area narrows as targets get closer.
    
*   **Understanding the Radar Screen:**
    
    *   **Top-Down View:** Radar display is a map of your surroundings.
    *   **Scale Settings:** Adjust scale using **F3** and **F4** keys.
    *   **Numbers Represent Altitude:**
        
        *   Top white number = maximum scanning height (e.g., 20,000 feet).
        *   Bottom blue number = minimum scanning height (e.g., 9,000 feet).
        
    

Controls:
---------

*   **A6, A3, A1:**
    
    *   Adjusts horizontal scan width (e.g., A6 scans widest, A1 the narrowest).
    
*   **4B, 2B, 1B:**
    
    *   Changes the number of vertical sweeps. Fewer sweeps (e.g., 1B) focus radar energy on a specific altitude range.
    

3\. **Target Identification and Locking**
-----------------------------------------

Key Features:
-------------

1.  **Soft Lock:**
    
    *   Performed by pressing **TMS up once**.
    *   Locks a target while continuing to scan for others.
    *   Provides key target data:
        
        *   **Altitude** (e.g., 20,000 feet).
        *   **Speed** (e.g., 345 knots).
        *   **Closure Rate:** Positive (+) = approaching; Negative (-) = moving away.
        *   **Direction:** Shown via a yellow line (longer line = faster speed).
        *   **Bullseye Data:** Displays bearing and distance from a reference point (bullseye).
        
    
2.  **Hard Lock:**
    
    *   Press **TMS up again** to fully lock a target.
    *   Devotes all radar energy to tracking that target, making tracking more precise but limiting scans of other areas.
    

4. **Air-to-Air Combat: Beyond Visual Range (BVR) Engagement**
---------------------------------------------------------------

Steps to Shoot a Missile:
-------------------------

1.  **Switch to Air-to-Air Mode:**
    
    *   Activate Master Arm and select missile type (AMRAAM = medium-range; Sidewinder = short-range) using **Enter**.
    
2.  **HUD Symbology:**
    
    *   **Maximum Range Triangle:** Shows the farthest distance a missile can reach.
    *   **Optimal Range Line:** Indicates where a missile has a high chance of reaching its target.
    *   **Guaranteed Kill Zone:** If the target is within this zone, the missile will hit regardless of evasive actions.
    
3.  **Center the Dot:**
    
    *   Align the smaller circle (shoot window) inside the larger HUD circle before firing.
    
4.  **Missile Firing Commands:**
    
    *   **FOX3:** Callout for firing an AMRAAM.
    *   **FOX2:** Callout for firing a Sidewinder.
    

5. **Dogfight Mode**
---------------------

When Missiles Aren’t an Option:
-------------------------------

1.  **Switch to Dogfight Mode:**
    
    *   Engage guns for close-range combat.
    
2.  **HUD Adjustments:**
    
    *   HUD provides targeting cues for engaging nearby enemies with the aircraft’s gun.
    
Study Guide for Transcript on F-16 Operations
---------------------------------------------

**Section 1: Dogfight Mode**
----------------------------

*   **Entering/Exiting Dogfight Mode**
    
    *   Press `D` to enter dogfight mode.
    *   Press `C` to exit and return to the last mode (NAV, air-to-air, etc.).
    
*   **Speed Management**
    
    *   Maintain **400 knots** during turns for maximum turn rate.
    *   If speed drops below 400 knots, ease off the turn and regain speed.
    
*   **Helmet Sight and Target Lock**
    
    *   Activate helmet sight with `Shift` + `Insert`.
    *   Use `TMS up long` to bring up a targeting circle and lock onto enemy aircraft.
    
*   **HUD Symbology**
    
    *   **Gun Cross**: Indicates where bullets will go when fired.
    *   **Gun Funnel**: Adjusts to help predict bullet trajectory relative to the target.
    *   Focus on the tracers' path to adjust shots, aiming slightly ahead of the moving target.
    
*   **Practical Tips**
    
    *   Avoid pulling fully back on the stick; prioritize maintaining 400 knots.
    *   Watch tracer rounds and "walk" them onto the target.
    

**Section 2: Laser-Guided Bombs (LGBs)**
----------------------------------------

*   **Preparation Steps**
    
    1.  Enter **Air-to-Ground Mode**: Press `Backspace`.
    2.  Navigate to Steer Point 5: Press `CZ` to center targeting systems.
    3.  Turn on the targeting pod by selecting the targeting pod button, then `Standby`, and `Air-to-Ground`.
    
*   **Target Acquisition**
    
    *   Make targeting pod the Sensor of Interest (SOI) by pressing `DMS down`.
    *   Enter `Area Mode` with `TMS up once`.
    *   Use the Field of View (FOV) cycle to zoom in and locate targets.
    
*   **Bomb Release Procedure**
    
    *   Ensure **Master Arm** and **Laser Arm** are on.
    *   Align the vertical flight line with the flight path marker for an accurate drop.
    *   Use the countdown timer to anticipate the release:
        
        *   **First Countdown (optional)**: Lob the bomb mid-pull.
        *   **Second Countdown**: Hold the weapon release button 5 seconds before the drop marker aligns.
        
    
*   **Laser Guidance**
    
    *   Maintain the targeting pod on the target after bomb release.
    *   The laser activates 6 seconds before impact, indicated by a flashing "L".
    *   The bomb self-guides to the target.
    

**Section 3: HARMS (Anti-Radiation Missiles)**
----------------------------------------------

*   **Activation and Setup**
    
    1.  Enter **Air-to-Ground Mode**: Press `Backspace`.
    2.  Power on HARMS via the SMS page.
    3.  Display the HAD (HARM Attack Display) on your left MFD.
    
*   **Understanding HAD Symbology**
    
    *   **Yellow/Green Numbers and Letters**:
        
        *   `2`, `3`, `U`, `S`: SA-2, SA-3, Unknown, Search radar sites.
        
    *   **Green**: Passive emissions; historical data.
    *   **Red**: Actively emitting (threat).
    
*   **Target Lock and Missile Firing**
    
    *   Use `DMS down` to make HAD the SOI.
    *   Lock onto radar sites (e.g., SA-3) with `TMS up`.
    *   Fire when the range carrot aligns inside the HUD box.
    *   **Callout**: "Magnum" indicates HARM launch.
    
*   **Missile Behavior**
    
    *   Fire-and-forget operation.
    *   SAM sites may turn off radar; consider flying toward the target to maintain threat.
    

**Section 4: Dumb Bombs (CCIP Mode)**
-------------------------------------

*   **Setup and Activation**
    
    1.  Enter **Air-to-Ground Mode**: Press `Backspace`.
    2.  Access SMS page to select dumb bombs (e.g., BD-33s).
    
    *   Configure bomb settings: single, pairs, ripple settings.
    *   Change mode to **CCIP** via MFD.
    
*   **HUD Symbology**
    
    *   **CCIP Line and Circle**: Predicts bomb impact point.
    
*   **Bomb Deployment**
    
    *   Aim the CCIP circle over the target.
    *   Press the weapon release button to drop.
    

**Section 5: Mavericks (Air-to-Ground Missiles)**
-------------------------------------------------

*   **Preparation and Spooling**
    
    1.  Enter **Air-to-Ground Mode**: Press `Backspace`.
    2.  Power on missiles (`Ds` and `Bs`) via the SMS page.
    3.  Wait ~3 minutes for missiles to align. Countdown displayed in the bottom left of the MFD.
    
*   **Air-to-Ground Radar Modes**
    
    *   **GM (Ground Moving)**: Detects stationary targets (e.g., buildings, parked vehicles).
    *   **GMT (Ground Moving Target)**: Detects only moving targets (e.g., columns, active vehicles).
    
*   **Target Engagement**
    
    *   Wait for Mavericks to spool up completely.
    *   Use radar or targeting systems to locate and lock targets.
    *   Once locked, fire and guide the missile as necessary.

This study guide is designed to help you thoroughly understand the material from the tutorial on Falcon BMS to prepare for a test. Below is a breakdown of the key concepts, procedures, and features covered in the material.

**1. Maverick Missile Deployment**
-----------------------------------

Key Steps:
----------

1.  **Mode Selection:**
    
    *   Begin in "Pre-mode."
    *   Switch to "Visual mode" by cycling through modes using the designated button.
    *   In "Visual mode," a square appears on the HUD, allowing targeting.
    
2.  **Uncaging the Maverick:**
    
    *   Press the "Uncage" button (default: "U" on the keyboard) to view the Maverick seeker head image.

    *   Press the weapon release button to fire the missile.

**5. Stores Configuration (Cat3 vs. Cat1)**
--------------------------------------------

*   **Cat3 (Default for Ground Ordnance):**
    
    *   Limits G-force to prevent damage to heavy ordnance (e.g., fuel tanks, air-to-ground munitions).
    
*   **Switch to Cat1:** After jettisoning heavy stores, switch to Cat1 for maximum G performance.
