Controls Setup
--------------

*   Map joystick axes for pitch and roll under Advanced > Flight Control
*   Map avionics controls under Advanced > Avionics Control
*   Use the default BMS key file as a starting point
*   Customize key bindings in the Controllers menu
*   Save custom key files with unique names

Game Modes and Navigation
-------------------------

*   Tactical Engagement: Pre-made scenarios for training
*   Campaign: Long-term gameplay with dynamic missions
*   Use the map screen to select flights and commit to missions
*   Accelerate game time using Shift+Tab (up to 64x speed)

Aircraft Startup Procedure
--------------------------

1.  Switch main power on
2.  Set engine feed to normal
3.  Set air source to normal
4.  Turn on jet fuel starter
5.  Move throttle to half, then use Alt+I to move past detent
6.  Bring throttle back to idle once engine starts
7.  Turn on avionics power switches
8.  Set INS to NORM (takes 8 minutes to align)
9.  Turn on sensor power (radar, altimeter, etc.)
10.  Turn on HUD and arm ejection seat
11.  Activate countermeasures and RWR
12.  Set up radio levels and transmit settings
13.  Load data cartridge with custom settings

Air-to-Air Combat
-----------------

*   Use backspace to cycle between NAV and A/A modes
*   Understand radar display and symbology
*   Soft lock targets with TMS up once
*   Hard lock targets with TMS up twice
*   Use AMRAAM (Fox 3) for beyond visual range engagements
*   Use Sidewinder (Fox 2) for closer engagements
*   Dogfight mode activated with 'D' key, canceled with 'C' key
*   Maintain 400 knots for optimal turn rate in dogfights

Air-to-Ground Combat
--------------------

*   LGB (Laser Guided Bomb) procedure
*   HARM (High-speed Anti-Radiation Missile) usage
*   CCIP (Continuously Computed Impact Point) for dumb bombs
*   Maverick missile employment

Campaign Mode
-------------

*   Choose between blue force advantage, neutral, or op force advantage
*   Set mission priorities or let HQ decide
*   Customize data cartridge settings (chaff/flare programs, MFD layouts)
*   Set loadouts for yourself and wingmen
*   Use the HSD (Horizontal Situation Display) for navigation
*   Communicate with AWACS using 'Q' menu
*   Understand CAT 1 and CAT 3 stores configurations

Key Concepts and Terms
----------------------

*   FCR: Fire Control Radar
*   HSD: Horizontal Situation Display
*   TMS: Target Management Switch
*   DMS: Display Management Switch
*   RWR: Radar Warning Receiver
*   INS: Inertial Navigation System
*   CCIP: Continuously Computed Impact Point
*   CCRP: Continuously Computed Release Point
*   BVR: Beyond Visual Range

Important Keyboard Commands
---------------------------

*   Backspace: Cycle between NAV and A/G modes
*   Enter: Cycle between missile types in A/A mode
*   D: Activate dogfight mode
*   C: Cancel dogfight mode
*   Q: Open communications menu
*   Shift+Tab: Accelerate game time
*   Alt+I: Move throttle past detent during startup
