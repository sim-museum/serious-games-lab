No additional libraries are needed for the SAT programs.

When you first run ./FreeFalcon.sh, Free Falcon will be unpacked and installed.
When you reach the final install screen, completing the FreeFalcon 6.0 Installation,
deselect the option Launch Free Falcon 6 Config Editor
Otherwise, simply select defaults until the installation is done.

The first time you run ./FreeFalcon.sh
it unpacks and creates many files in the $HOME directory.
In case this is of concern, a list of the created files follows.
