The Map
The campaign map is capable of displaying the entire area of the Battle of Britain campaign, from northern
England to northern France, based on original Ordnance Survey data from the period. The large scale map can
be displayed at 4 major levels of magnification, from 400 miles square down to a 50 mile square area. At a
finer level of detail the map becomes a continuously scaleable down to a scale of 2 miles square. The zoom
level is easily adjustable, and the map may be scrolled, either using the scroll bars at the right and bottom
sides of the map, or by clicking and dragging with the left mouse button while the pointer is over the map
portion of the screen.

As well as the campaign map, the campaign screen has a set of icons in the top right and five toolbars:

                 TimeToolbar

                 Main Toolbar

                 Map Toolbar

                 Scale Toolbar

                 Teletype Toolbar

The fixed set of icons in the top right of the screen offer the following functions:

      • Quit
        Return to the Main Menu.
      • Toolbar Control
        This gives access to the toolbar dialogue box. This box allows the player to select which of the five
        toolbars will be displayed on the screen at any one time.
      • Size

Toggle between full screen and maximized displays.

Right Click
Right click on the mouse anywhere apart from the campaign map on the campaign screen and a menu system
that duplicates the functionality of the toolbars will open.

Scale Toolbar
The units shown on the scale are either imperial or metric depending on the selection under the units option in
the Sim Config menu. If imperial units are selected then the scale is displayed in nautical miles. Kilometres
are used if metric units are selected.

The scale is fully dockable so that you can move it around the map. The scale can be forced to display
horizontally or vertically. If the scale is currently vertical, then click and hold on the top portion of the scale
above the origin and drag it to the desired location.

One method of altering the map scale is to click and hold with your left mouse button over the map toolbar.
Select a location at the bottom of the scale and move the mouse upwards to decrease the map resolution.
Select a location at the top of the scale and move the mouse downwards to increase the map resolution.

The time toolbar is located at the bottom left hand side of the screen.

The Teletype section of Time Toolbar shows the current date, time and acceleration rate of the game. Click in

The control icons below the teletype section determine the flow of the campaign:

Game Speed Icon:

opens the Time Control dialogue box.

Pause Icon:

pauses the game.

Play Icon:

plays the game in real time, where one game second takes one real second (this is the default setting).

Accel Icon:

the game will run at the accelerated rate determined by the settings in the Time Control dialogue. The
accelerated time can range from x0, when the game is paused, through x1, which indicates that the game is
running in real-time, to x600. When the accelerated time shows x600, the game is progressing 600 times faster
than real time. So for every second of real time the game will attempt to progress by 10 campaign minutes.

Aircraft Allocation

The aircraft allocation dialogue box shows where the nation's current aircraft production is destined, and what
is required. There is a line of information for each type of aircraft under production:


      • Aircraft: shows the type of aircraft produced.
      • Squadron/Gruppe: shows the destination of current production.
      • Airfield: shows the home of the above squadron.
      • Rdy(Ttl): shows the number of combat-ready aircraft that the above squadron has at present.
        Followed, in brackets, by the total aircraft, including those under repair.
      • Req.: shows the total number of combat ready aircraft required.
      • ETA: shows the estimated time of arrival of the next production aircraft.
      • Rate/wk: the current production capacity in aircraft produced per week.


Aircraft allocation can be a completely automatic process. The game AI will choose the next squadron or
gruppe to receive aircraft, based on an assessment of the overall status. The automatic system allocates
aircraft in blocks of up to four.
However, manual intervention is possible. The squadron/geschwader box allows you to change the unit that
will receive the next allocation, and it is also possible to change the number of aircraft required. Note that for
the Luftwaffe, Ju87 aircraft are never replenished.

Control Dialogue
The control dialogue box is displayed when the game speed icon on the time control toolbar is clicked. It
affords the player a high degree of control over the way in which the gameplay progresses. The acceleration
page allows the player to define how the game speed varies throughout the game; the fly page determines
when the player will be offered the chance to get into an aircraft during the campaign.

Accel

When the Accel tab is set, the top half of this dialogue box is used to define the accelerated rates of gameplay
under the following conditions:

      • Non-raid
      • Reconn(aissance)
      • Raid
      • Dialogue

So, if the dialogue rate is set to 50, then the game will play 50 times faster than real time when a dialogue box
is displayed and the Accel Icon on the Time Toolbar has been pressed. The other three options refer to the
current state of Luftwaffe activity - a raid is determined to be in place once it has been detected and confirmed
by Radar or the Observer Corps. The current activity is shown in red.

The bottom half of the dialogue box details the conditions that can be set for the game to revert to normal time
from accelerated time. For instance if the "Small Engagement" option is ticked then the game will revert to
normal whenever a "small" engagement occurs. A small engagement is defined as one that consists of two or
less Staffeln or Squadrons.

      • Raid Forming

         when a Luftwaffe raid first gets in detection range of the RAF

      • Scramble

         when any planes on your side take off

      • Reach Coast

        when any Luftwaffe aircraft cross the English coast.

      • Small Engagement

         when a raid of two or less Staffeln are engaged by RAF fighters

      • Large Engagement

         when more than two Staffeln are engaged by RAF fighters

      • Losses

         when either side suffers an aircraft loss.

      • Ground Attack

         when a Luftwaffe ground attack mission reaches it s target area

Dossier
All of the major facilities in the south-east of Britain have an associated dossier which contains relevant
information about its current known status. You can access the dossier of any facility displayed on the main
campaign map (see the small filled coloured circles) by clicking on it, or by clicking on the 'details' icon at the
bottom of the asset/target list which is accessed from the main toolbar. An airfield dossier can also contain
squadron details when playing on the RAF side - see further information in the RAF dialogues section.

The dossier consists of two tabs: General and Damage.

Under the General tab you will receive specific information as follows:

      • Name
         The name of the facility.
      • Type
         The type of facility - for example the type of radio direction finding facility.
      • AAA
         A description of whether or not there has been AAA activity near this facility.
      • Balloon
         A description of whether or not there has been barrage balloon activity above this facility.
      • Status/Damage

 The current damage status and a description of specific problems.

When taking the role of the Luftwaffe Commander, you will also be given any relevant reconnaissance
information.

Click on the Damage tab to display further damage status information. A list box allows the player to display
all of the elements that make up the particular location or asset. The fraction displayed to the right of the name
in the list box indicates the number of items making up the asset in the elements and the number damaged. For
instance 1/4 means that there are four items in the element and one item has been damaged.

Buttons

The buttons at the bottom of the dossier box are as follows:

      • Centre:
        Move the map so that the target is centred on the screen.
      • Zoom:
        Centre the map and simultaneously zoom in one level.
      • Photo:
        Display a 3D image of the target. (Only for RAF command.)
      • Patrol:
        Set up a patrol to the identified target. (Only for RAF command)
      • Reconn:
        Set up a reconnaissance mission to the identified target. (Only for Luftwaffe command.)
      • Authorize:

Authorize a strike against the selected target. (Only for Luftwaffe command.) The resulting mission type will
be the same as the last mission whose task list has been viewed or edited in the mission folder. The mission
can be edited from within the mission folder at any stage before take-off time, and will be initially scheduled
for thirty minutes after the time of authorisation.

Frag
The frag button is on the lower left corner of the mission folder box, and should be used when you wish to fly
in one of the displayed missions. The button will only be active when you are allowed to fly in the selected
group you cannot fly in groups that are preparing to takeoff or have landed.

 Select a mission task line in the mission folder and click the FRAG button to view the FRAG screen. You can
also access this screen by clicking the FLY button on any TAKE-OVER warning message that appears.

The top of the FRAG screen lists the staffel / squadron units taking part in the mission. Below this list is a
bank of pilot slots corresponding to the current selected unit. The arrangement of the pilot slots shows the
flight formation that will be used in the air. Your player name should appear automatically in the lead pilot
slot. You can change your position within the squadron / staffel by clicking another pilot slot. You can view
other units by clicking different unit lines (if there are any) and choose to fly in them by clicking any pilot
slot.

If you lose track of your current pilot selection then hit the Return to Player tickbox. The unit containing
your pilot will be selected and displayed.
The term frag is shorthand for the fragment of the orders in the current mission that concern the squadron or
staffeln in which you are to fly.

RAF Analogue Clock
The RAF clock shows the current time on a standard 12-hour display. A set of coloured triangular shapes are
arranged around the edge of the clock. During the Battle of Britain (and throughout the entire war) the
Luftwaffe time was in advance of the RAF s by one hour. To avoid confusion we are using RAF time
throughout.

The first five minutes after the hour is coded red, the second five minutes is yellow and the third is blue. The
code then repeats so that from 15 to 20 minutes the code is red again.

Whenever a Luftwaffe direction plot is displayed on the map, its arrow colour will represent the time that the
information was provided. In this way it is possible to instantly recognise the age of the plots on the map.

The route button is on the lower part of the mission folder. Click on the route button to display the route
dialogue box, which details all aspects of the route covered by the group selected in the mission folder.

There are five tabs at the top of the route box when playing under Luftwaffe command, and only three tabs in

The major column headings identify the type of waypoint, and three major parameters:

        controlled pilots enter their landing pattern.
      • ETA
           The estimated time of arrival at the waypoint.
      • Rel to ToT

 The time relative to the estimated time over the target.

Waypoint
The waypoint dialogue is displayed when a waypoint is highlighted in the routes dialogue box, or on the main
campaign map. Waypoints are displayed as unfilled circles on the main map.

The altitude of any waypoint can be edited, and it is also possible to alter the arrival time at the Initial Point
waypoint. To find out more about the Initial Point waypoint see the routes dialogue description. The
parameters listed on the waypoint dialogue box are:

      • Alt
        The altitude of the waypoint, which can be altered using the pull down box.
      • ETA
        The estimated time of arrival at the waypoint. Only the ETA for the Initial Point waypoint can be
        edited using up and down arrows.
      • Rel to ToT
        The expected time of arrival at the waypoint relative to the time over the target. A negative value
        means a time before the target is reached.
      • Fuel
        This is the estimated fuel remaining when the waypoint is reached, defined in terms of minutes of
        flying time still possible.
      • Escort Fuel
        Where relevant, this shows the amount of available fuel left in escorting planes by the time this
        waypoint is reached, expressed in available minutes of flying time.
      • Bearing
        The compass bearing from the previous waypoint.
      • Range

The distance from the previous waypoint.

There are six buttons at the bottom of the waypoint box:

      • Prev
          Takes you back to the previous waypoint in the current route.
      • Next
        Takes you to the next waypoint in the current route.
      • Route
          Displays the route dialogue box which contains the current waypoint.
      • Insert Creates a new waypoint midway between the current and the next waypoint. Groups of
        aircraft on a mission can diverge after the 'initial point'. Hence, if the current waypoint is the 'initial
        point' then a new waypoint is inserted for each group of aircraft.

      • Delete

 Deletes the currently selected waypoint. This facility is only available for inserted waypoints.

Centre

 Centres the selected waypoint on the main campaign map.

Weather
The weather information box is accessed by clicking on the weather icon on the main toolbar. It shows the
days' weather forecast for the south-east England.

Conditions

The conditions are summarised, with an eye kept on visibility. Your aircraft are not fitted with sophisticated
radar equipment and so target acquisition is by eye. In bad conditions, it will be difficult to identify targets.

Temperature and Pressure

For fighter aircraft, this information has no direct bearing. However the data does give an indication of
weather to be expected.

Cloud Base

The altitude of the cloud base is displayed next to the cloud coverage. The cloud coverage is presented as a
fraction. 8/8 represents total coverage and 0/8 represents a clear sky.

Wind Speed and Direction

Wind speed and direction are quoted at two different altitudes.

Gusts

If the weather is changeable, it is likely that you will suffer from gusts. This will make flying more stressful as
you have to constantly adjust the controls.

July 10th 1940
    1. You lead RAF Fighter Command charged with the duty of defending the air space over the United
       Kingdom. Specifically you must defend your country from the invasion force building up on the
       continent of Europe. You must ensure that Fighter Command remains an effective force throughout
       the summer until mid September when bad weather eliminates the risk of invasion.
    2. You have 53 fighter squadrons at your disposal: 34 have Hurricanes and the rest are equipped with
       Spitfires. All told, there are over 600 aircraft in your command. You are limited to scheduling
       missions for a maximum of 32 squadrons at any one time.
    3. In pursuit of your campaign objectives, you will respond to threats as they appear on the map display.
       This map shows the result of collating intelligence about enemy movements from various sources
       including RDF and the ROC.
    4. It is possible to define both patrol and intercept missions. Patrols can be vectored to threats when they
       are in the air. You can either design a mission from scratch personally or just set the overall
       parameters and let your staff complete the details.
    5. You are cleared to fly missions in Spitfire and Hurricane fighters.

RAF Organisation
Fighter Command consists of four Groups: 10, 11, 12 and 13. Each Group is assigned a geographical area
rather than a specific set of squadrons. Squadrons move from Group to Group to fulfil the objectives of
Fighter Command.

Although a squadron can have more pilots and aircraft assigned to it, a complete squadron in the air consists
of twelve aircraft. The squadron is split up into two flights of six and each fight consists of two sections of
three.

RDF and ROC
Most of the intelligence about enemy movements is gathered by either Radio Direction Finding (RDF)
methods or the Royal Observer Corps (ROC).

RDF methods are now universally referred to as radar. Fighter Commands had two coastal systems in
operation during the Battle of Britain. The Chain Home system was capable of detecting aircraft at a distance
of up to about 150 kilometres (90 miles). At about 15, 000 feet the detection range was about 100 kilometres(
60 miles).However CH was not very good at detecting aircraft at low altitude. The Chain Home Low system
was developed to overcome this limitation. This type of radar had a range of about 30 kilometres ( 20 miles)
at low altitude.

Neither system could be used to track aircraft once they had passed by the system and flown inland. The job
of detecting aircraft over England fell to the Royal Observer Corps. Groups of two or three people were
stationed all over the south east of England and they reported all aircraft movements to a central station. This
system worked remarkably well - however heavy cloud did have a large effect on its effectiveness.

All the information generated passed through a filtering stage so that the information presented on the map
was consistent and as error free as possible.

Routes Overview
A route is a set of waypoints. Initially, each route consists of the same standard set of waypoints:

      • Take Off

Generally, the position of the airfield from which the aircraft take off.

      • Fighter Rendezvous

The location at which fighters from different airfields join up.

      • Dogleg

An extra waypoint that allows fine control of the direction that the mission approaches the initial point.

      • Initial Point
        This is the entry point to the target area. The units split at this point to perform their individual duties.
      • Estimated Point

This is the estimated location of the hostile at the time the intercept takes place.

      • Target Egress

The point of exit from the combat area air space.

      • Fighter Dispersal

The point at which fighters from different airfields split up.

      • Landing

The position of the landing airfield.

Aircraft in the same mission fly always together from the Fighter Rendezvous to the Initial Point. Each target
has its own set of waypoints in the Target Zone. Units follow the waypoints associated with their own target.

It is possible to insert and delete additional waypoints. However it is not possible to delete any of the standard
set. Additional waypoints can be added to both the main route and target zone.

Luftwaffe Organisation
The Luftwaffe organisation bears no resemblance to that used by the RAF. It is confusing then to use RAF
terms like wing and squadron when discussing Luftwaffe aircraft units.

Two of the Luftwaffe s three Luftflotten are involved in the Battle:

      • Luftflotte 2: HQ in Brussels
      • Luftflotte 3: HQ in Paris

Each Luftflotte consists of a number of Geschwader. A Geschwader is a unit of about one hundred aircraft all
of the same type. Each Geschwader consists of up to 3 Gruppen and each Gruppe is made up of three Staffeln.
Staffeln of aircraft are further divided depending on the type:

      • A complete fighter Staffel will have three Schwarme of four aircraft, making twelve aircraft in total.
        Each Schwarme consists of two Rotte or leader-wingman pairs. A Schwarme is equivalent to the
        finger four used by many airforces since the Second World War. Luftwaffe pilots are generally
        credited with inventing this very successful formation.

      • Bombers will fly in Kette of threes or fives in a V formation. A bomber staffel can consist of 9, 10, or
        12 aircraft depending how many serviceable aircraft the gruppe can muster (27, 30, 36 respectively).
        A very strong bomber staffel may be able to field 40 bombers which will fly in a box of 4 staffel of 2
        kette 5's.

The plural of Gruppe is Gruppen and the plural of Staffel is Staffeln.

Fighter Command consists of four Groups: 10, 11, 12 and 13. Each Group is assigned a geographical area

RAF Defensive System
The RAF have built radio masts on the English coast. Their purpose is unclear, however there is a strong
likelihood that they form some sort of aircraft detection system. The RAF are able to intercept raids

Routes Overview
A route is a set of waypoints. Initially, each route consists of the same standard set of waypoints:

      • Take Off
        Generally, the position of the airfield from which the aircraft take off.
      • Bomber Rendezvous

The location at which bombers from different airfields form up.

      • Fighter Rendezvous

The location at which fighters from different airfields form up.

      • Dogleg
        An extra waypoint that allows fine control of the direction that the mission approaches the initial
        point. The dogleg can deceive the defenders into thinking your target is on a course away from your
        intended target.
      • Initial Point

This is the entry point to the target area. The units split at this point to perform their individual duties. In the
target area, bombers will be looking closely for the target and making minor adjustments to their course. The

      • Target Egress
        The point where aircraft fly towards in order to clear the target after dropping their bombs.
      • Fighter Dispersal

The point at which fighters from different airfields split up.

      • Bomber Dispersal

The point at which bombers from different airfields split up.

      • Landing

The position of the landing airfield.

Aircraft in the same mission fly together from the Rendezvous to the Initial Point. Each target has its own set
of waypoints in the Target Zone. Units follow the waypoints associated with their own target.

It is possible to insert and delete additional waypoints. However it is not possible to delete any of the standard
set. Additional waypoints can be added to both the main route and Target Zone.

Directives
The Directives Dialogue displays the controls for the automatic mission builder. The dialogue consists of four
columns of options above a Resource Allocation Table that defines the usage of the available aircraft.

Bomber Allocation

There are three planning periods per day: Morning, Mid-day and Afternoon.

You can use the spin buttons to specify how the available daily resource can be apportioned throughout the
day. For instance, it is possible to allocate the whole aircraft resource to just one session by increasing the
percentage for that period to one hundred. However there is a maximum Staffel usage in any one period which
is set at 96 - 32 gruppen. In the latter parts of the battle you will find that the spin buttons will be limited to
percentages below one hundred. This will be because higher percentages would exceed the 96 Staffel limit.

The Luftwaffe system is based firmly on the number three: three Staffel in a Gruppe and three Gruppe in a
Geschwader. For this reason, when resources are scarce, say only 1 Geschwader is available, then 30% is
interpreted as a request for one third of the resources.

JU87 Allocation

In the Convoy campaign, the JU87 dive bomber is the main bombing aircraft, and so you will want your JU87
aircraft allocated in the same percentages as everything else (maintain %). If you want your Stuka force to be
counted when reserving forces for each period then use the Maintain option. The None option means
that Stukas are not used at all by the directives. Morning , Midday and Afternoon options reserve all
available Stuka squadrons for that period but are not counted as part of the % for the other periods.

In those periods when the JU87s are disabled, a higher number of level bombers and fighters will be allowed
up to the 96 staffel limit, allowing larger level bombing raids to be formed.

Mission Timing

Missions within a session can either be consecutive or co-incidental.

When co-incidental is selected, all the missions for the period will be planned to come over their target areas
at the same time. When consecutive is selected, missions are planned so that they come over the target area at
different times. The missions target times will, more or less, be evenly spaced through the session.

will perform a strafing run over the target. Losing altitude makes the fighters more vulnerable, but if they
successfully cause the RAF to abandon their guns, it is a useful procedure.
 • %tied
The percentage of fighter escorts that are closely attached to the bombers allocated in the same row.
 • %free
The percentage of fighter escorts that are loosely attached to the bombers allocated in the same row. You will
note that it is possible to select the %tied and %free values to add up to less then 100%. This is because there
is a third type of escort called "Returning". This is the escort that sets off much later than the main force so
that it is available to escort the returning bombers. When "%tied" and "%free" do not add up to 100%, then
the rest is allocated to "returning" duty. The return escort will pick up the bombers on their return leg, offering
some level of protection.

Mission Folder
The mission folder dialogue lists all the raids that have been planned during the current session. A raid is only
removed from the list at the end of the day. The contents of the list can be controlled using the set of filters at
the top of the dialogue. Strike, escort, sweep and reconn refer to mission types. Planned, active and landed
refer to whether or not the mission is yet to takeoff, is in transit or completed. Take the tick off landed, for
instance, in order to remove all missions that have been completed from the list.

Each duty type within a raid is allocated one row in the list. This means that the strike and escort information
is presented separately:

      • Raid

This is the number allocated to the raid, for example R201. R stands for raid and a raid is always allocated a
three-digit number. The first digit represents the hour when the raid was initiated. So R601 is the first raid
launched after 6 o clock.

        For each duty, the following information is presented:

      • Size
        This is the number of aircraft allocated to the duty.
      • Type
        This is the type of duty. Strike duties include Dive Bombing and Level Bombing. Escorts can be
        Attached, Detached or Returning. An attached escort must stay close to the bombers and a detached
        escort can roam freely. A returning escort times take-off so that it can arrive over the return route in
        time to escort the strike aircraft as they return home.
      • T/O
        This is the take-off time.
      • ToT
          This is the estimated time over the target.
      • Status
        This is the operational status of the aircraft.
      • Target
        This is the target for the raid.
      • RAF
        This is the number of RAF that are known to be intercepting.
      • Tally

The positive number represents the number of unconfirmed kills and the negative number represents the
number of losses.

Review
The review summary box is displayed on clicking the Review icon on the main toolbar. The review summary
shows all of the major information about the campaign and the known state of RAF and Luftwaffe resources.
The information is collated under five main categories of data, which are available through the five tags at the
top of the box:

Strength

The strength tag summarises the current overall strength of the Luftwaffe. The available aircraft are listed
according to type, and categorised according to their mission readiness, both as gruppen and individual
aircraft.

Aircraft

The aircraft tag gives a much more detailed summary of the overall aircraft status, including the current
production schedules. The information presented for all Luftwaffe aircraft types is listed as:

      • B/F Combat Ready
        The number of combat ready aircraft at the start of the campaign.
      • Damaged
             The number of aircraft that are currently damaged.
      • Repaired
             The number of aircraft that have been repaired during the campaign.
      • Destroyed
        The number of aircraft that can no longer be used in the campaign.
      • New
             The number of aircraft that have been manufactured.
      • C/F Combat Ready
        The current number of active aircraft.
      • Factories Max Production
        The maximum production rate of all aircraft factories in Germany per week. This assumes no damage
        to aircraft production sites, and maximum worker morale.
      • Factories Actual Production
          The actual production rate per week of factories
      • Aircraft in system
           The number of aircraft that are currently being repaired
      • Max. Repair Rate
           The maximum repair rate of aircraft per week, assuming no damage to repair facilities and the
        highest morale.
      • Actual Repair Rate

   The current repair rate of aircraft per week.

Targets

The assets tag summarises the state of important British targets. Each target is categorised according to its
damage status - either Unknown (in the case that there is limited intelligence available), OK, Damaged and
Destroyed. Each category of damage is listed according to the number of individual units that fit that damage
category, followed in brackets by the percentage of the total asset it represents. For instance 24 (80%) under
the 'OK' category for 'All Airfields' means that there are 24 airfields that have not suffered damage and that
this represents 80% of the total number of airfields.

The targets are categorised according to:

Task
The task dialogue is displayed when the task button is clicked for a mission highlighted in the mission folder.
It consists of five tabs: Summary, Bombers, Tied, Free and Return. The last three tabs are concerned with the
escorts and have similar functionality.

The dialogue is used to change the number and type of aircraft allocated to the mission. In addition it is
possible to change attack methods, escort positions and formations. It is also possible to instruct the escort to
strafe the target.

       • Summary

        This dialogue shows the current allocation of aircraft to the raids. It is possible to use the spin buttons
        to alter the bomber and fighter types and numbers for the raid. If changes are made, then the gruppen
        involved will be assigned automatically.

       • Bombers

        If you want to be specific about which Gruppe are allocated to the Strike Duty then it is necessary to
        go to the Bomber tag.

        At the bottom of the dialogue there is a list with the headings UNIT, TARGET, STATUS. Click on
        the existing Unit or 'Next Available' under the Unit heading to open the Unit dialogue.

        Click on the item in the Target column to change the current target for that particular unit of aircraft.

       • Escort

        The three escort tabs have similar functionality. ESCORT refers to close escort fighters, FREE refers
        to detached escort fighters, and RETURN is reserved for returning escort fighters. The Gruppe
        allocated to escort the raid can be changed in the same way as on the Bomber tab.

        The escort position options are the same as those on the Directives dialogue.

        If the strafe tick box is ticked then the fighters will attempt a strafing run over the target.

        The individual fighter task lists allow the user to alter the attack characteristics of the fighters
        involved mission, and the squadrons or targets involved:

       • Fighters Allocated
          Displays the number of aircraft selected for the mission.
       • Strafe
             With this box ticked, the fighters are likely to straffe their target.
       • Formation

The escort formation can be chosen from: Echelon Left, Echelon Right, Line Abreast, Line Astern, Diamond,
Box.

       • Escort Position

   The preferred position relative to the bombers can be selected from: Behind, Above, Port, Starboard, In
Front, Below.

       • Unit Type

In the lower section of the fighter task list there is a list of the squadrons that are currently assigned to the
