Near /Far horizon settings 

One of the main improvements that Bob2 brought to the table was the fartiles function which renders the terrain tiles in the distance.

In Rowanbob the near horizon is claustrophobic in bad weather and the far horizon setting in good weather renders just flat colour tiles in the distance.

 A compromise can be reached and I've included the settings in the BDG.TXT file provided. Unfortunately when  toggling the map the horizon is reset to the default far setting.
For this reason I use 'R' rather than the map to issue radio commands.  


Refer to Patch manual 0.98 section "2.1.2.27" for further info.

SINGLE PLAYER OR squadron  CAMPAIGN NOTES:

There is no single pilot as we know it in bob2, but by using directives to micromanage squadrons we can play a campaign lite version and still play a psuedo single player campaign, pretty easy once you get the hang of it.


Start a campaign. 
Set directives for squadron deploment.
Frag into squadron to fly as individual pilot as and when  you see fit.

Read the 0.98 patch manual section "3.3.3" for more info.
