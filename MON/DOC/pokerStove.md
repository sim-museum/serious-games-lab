PokerStove Basics
-----------------

*   PokerStove can be downloaded from PokerStove.com
*   Used for pre-session and post-session analysis
*   Helps study equities and ranges
*   Assists in figuring out optimal or near-optimal plays

Using PokerStove
----------------

*   Player 1: Enter your specific hand
*   Player 2: Enter a range of hands or use "R" for random
*   Select board cards for post-flop analysis
*   Use "Evaluate" to calculate equities
*   Results show win percentage, tie percentage, and overall equity
*   Can input dead cards for live game analysis
*   Two evaluation methods: Enumerate and Monte Carlo

Advanced Features
-----------------

*   Preset hand ranges: Any pair, any broadway, any suited, all
*   Slider bar to adjust range percentages
*   Can deselect suits for specific flush draw scenarios

StocksPokerCombo
----------------

*   Similar to PokerStove but with additional features
*   Can be downloaded from StocksPoker.com
*   Useful for post-session and pre-session analysis

Using StocksPokerCombo
----------------------

*   Hand picker similar to PokerStove
*   Can select top or bottom percentage of hands
*   Allows for more detailed continuance range analysis

Range Analysis
--------------

*   Calculate opponent's stealing range (e.g., 30% of hands)
*   Determine continuation range (e.g., 10s+, AQo+)
*   Calculate fold frequency and profitability of 3-betting

Variable Effect
---------------

*   Account for miscalculations in opponent's range
*   Add extra hands to balance out incorrect assumptions
*   More aggressive opponents require wider variable effect

Weighted Ranges
---------------

*   Assign percentages to different range possibilities
*   Calculate overall equity based on weighted probabilities
*   Useful when you have concrete information about opponent's tendencies

Equity Calculations
-------------------

*   Use the formula: Profit = (Pot size × Win%) - (Bet size × Lose%)
*   Consider fold equity and potential to improve when behind

Key Concepts
------------

*   Range merging
*   Continuation ranges
*   Break-even points for 3-betting and 4-betting
*   Stack-to-pot ratio (SPR) and its impact on decision-making

Important Formulas and Calculations
-----------------------------------

*   Fold equity calculations
*   Weighted range equity calculations
*   Break-even point for bluffs and semi-bluffs

Software Tips
-------------

*   Keep a notepad and calculator handy for quick calculations
*   Save calculations for future reference
*   Use both PokerStove and StocksPokerCombo in conjunction for comprehensive analysis

Strategic Considerations
------------------------

*   Adjust ranges based on opponent type (TAG, LAG, etc.)
*   Consider position when analyzing ranges
*   Factor in stack sizes and SPR when making decisions

Common Mistakes to Avoid
------------------------

*   Overestimating opponent's continuation range
*   Failing to account for variable effect
*   Ignoring position and stack sizes in range analysis

Practice Exercises
------------------

*   Calculate equities for common preflop and postflop scenarios
*   Analyze 3-bet and 4-bet situations using both software tools
*   Create weighted ranges for different opponent types

Remember to practice using both PokerStove and StocksPokerCombo regularly to improve your range analysis and decision-making skills in poker.
